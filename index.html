Sobre mí

Desde que mi curiosidad me llevó al mundo de la informática, he estado en un constante proceso de aprendizaje. A través de proyectos personales y experiencias educativas, he adquirido habilidades en programación, administración de bases de datos, desarrollo web, entre otros campos. Es por ello que mi objetivo principal es combinar mi formación académica con el continuo autoaprendizaje para resolver problemas y crear soluciones innovadoras.

Mis Proyectos

CALCULADORA

Calculadora desarrollada con HTML, CSS y JavaScript es una herramienta web rápida y responsive. Realiza operaciones matemáticas esenciales de manera eficiente y se adapta a todos los dispositivos sin importar su tamaño.

TO-DO LIST

Aplicación simple y efectiva desarrollada con HTML, CSS y JavaScript que te ayudará a mantener un seguimiento de tus actividades. Te permitirá agregar, editar y marcar tareas completadas de manera rápida.

SONGS RECOMMENDER

Aplicación desarrollada en JAVA que permite al usuario de una manera cómoda obtener una lista de una o varias canciones del mismo estilo que la canción o género que haya escogido en el programa.

BANK APPLICATION

Aplicación orientada a un banco. Se encarga de gestionar y almacenar información de los clientes, incluyendo su correspondiente saldo. La interfaz gráfica permite a los usuarios inciar sesión, realizar diversas consultas y acceder a muchas mas funcionalidades.
